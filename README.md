# Smart home app
  React-native app to manage smart home devices integrated with [smart-home-sever](https://github.com/jonny22094/smart-home-sever)
  
  ### Tech stack
    - React-native
    - Expo
    - Typescript
    - Styled-components
    - React-native-color-picker
    - Socket.io
